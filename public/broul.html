Our Financial Data services

Accounts API

Capture all relevant information from customer bank accounts.

Returns IBAN, account number, account types, currency and more.

Balances API

Get a real-time view of available balance in customer bank accounts.

Refreshes in real-time or per API call set up by client.

Transactions API

Access a clean and standardized view of customers’ banking transactions.

Returns transactions for the past three years.

Beneficiaries API

Capture a list of all the beneficiaries linked to a connected account.

Returns an array of the beneficiary name, bank name, IBAN etc.

Scheduled Payments API

Capture a list of all scheduled payments set up on a connected account.

Returns an array of scheduled payments, the amounts and the payment dates.

Standing Orders API

Capture a list of all standing orders set up on a connected account.

Returns an array of standing orders, the amounts, and the payment dates.

Direct Debits API

Capture a list of all Direct Debits set up on a connected account.

Returns an array of direct debits, the amounts and the payment dates.

How does it work?

Customer connects their account
Customer provides permissions and enters their credentials
You call the accounts API to access their account details
You call the transactions API to access their account transactions
You call the balance API to view their real time account balance
